# All-in-One-View
Spring Boot를 활용하여 JPA를 통해 몽고DB로 사이트 방문자 정보를 수집합니다.

프론트단과 백엔드에서 API를 호출하며 실시간 데이터로 각종현황을 보여줍니다.
